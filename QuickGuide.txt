Simple Wizard — Getting started

Quick start (setup your own wizard build installer):

1. Unzip the distribution and rename the folder to the addon ID you want (example: plugin.program.mywizard).

2. Edit `addon.xml`: set the `id` to match the folder name and update `name`, `provider-name` and `summary`.

3. Replace `resources/icon.png` and `resources/fanart.jpg` with your artwork (keep filenames).

4. Edit the builds file: use `resources/texts/builds.xml` (XML) or `resources/texts/builds.json` (JSON). A sample is provided at `resources/texts/builds.example.xml` — keep the element structure the same.

5. Edit `uservar.py` and set `BUILD_URL` (or equivalent variable) to the public URL where you host your builds file.

6. Optional: pick an advancedsettings preset from `resources/advancedsettings/` (nexus|omega|piers) and host/use it as needed.

7. Zip the folder and install the addon in Kodi via "Install from zip file".

Notes:
- The wizard reads the remote builds file and presents installers; the builds file must be reachable over HTTP(S).
- Keep example files in `resources/texts/` as templates.
- For contributors: see `README.md` for more details and examples.
